# Customization Checklist

Use this checklist to customize your portfolio before deploying to GitHub Pages.

## 🎯 Essential Changes (Do These First!)

### Basic Information
- [ ] Update your name in navigation and hero section
- [ ] Update your title/tagline
- [ ] Update your bio in the About section
- [ ] Add your actual skills to the skills section

### Images
- [ ] Add portfolio project images to `/images/` folder
- [ ] Update image file names in the HTML (project1.jpg, project2.jpg, etc.)
- [ ] Replace placeholder image paths in `index.html`

### Portfolio Projects
- [ ] Update all project titles
- [ ] Update all project descriptions
- [ ] Add real project links (if available)
- [ ] Update four projects or add/remove more

### Contact Section
- [ ] Replace email address in `script.js` (search for "your-email@example.com")
- [ ] Update all social media links with your actual profiles

## 📋 Optional Enhancements

### Content
- [ ] Add more detailed project descriptions
- [ ] Add more portfolio items (duplicate portfolio-item divs)
- [ ] Expand the About section with more details
- [ ] Add links to live projects or GitHub repos

### Visual Customization
- [ ] Change primary color in `style.css` (--primary-color)
- [ ] Change secondary color in `style.css` (--secondary-color)
- [ ] Add a profile photo to the About section
- [ ] Customize fonts (change font-family in style.css)

### Functionality
- [ ] Setup proper email form (Formspree, Basin, etc.)
- [ ] Add analytics (Google Analytics)
- [ ] Improve SEO (meta descriptions, keywords)
- [ ] Add a blog section (optional)

### Performance
- [ ] Optimize and compress images
- [ ] Minify CSS and JavaScript (optional)
- [ ] Add favicons (`favicon.ico`)
- [ ] Improve load time

## 🚀 Before Going Live

- [ ] Test all links work
- [ ] Test on mobile devices
- [ ] Check form submission
- [ ] Test on different browsers
- [ ] Verify image paths are correct
- [ ] Remove placeholder text
- [ ] Update page title in `index.html`
- [ ] Add meta description to `index.html`

## 📝 Files to Edit

| File | What to Change |
|------|---|
| `index.html` | Names, content, links, image paths |
| `style.css` | Colors, fonts, spacing |
| `script.js` | Email address for contact form |
| `README.md` | Project overview |
| `/images/` | Replace with your actual images |

## 🎨 Quick Color Reference

Common color palettes (update `--primary-color` and `--secondary-color` in `style.css`):

**Blue & Purple** (Current)
```css
--primary-color: #667eea;
--secondary-color: #764ba2;
```

**Orange & Pink**
```css
--primary-color: #ff6b6b;
--secondary-color: #ff8787;
```

**Green & Teal**
```css
--primary-color: #11998e;
--secondary-color: #38ef7d;
```

**Dark Blue & Light Blue**
```css
--primary-color: #0066ff;
--secondary-color: #00d4ff;
```

## ✅ Deployment Checklist

- [ ] All customizations complete
- [ ] All images replaced
- [ ] All links updated
- [ ] Tested locally
- [ ] Git repository created
- [ ] Code committed with message
- [ ] Pushed to GitHub
- [ ] GitHub Pages enabled in repository settings
- [ ] Site is live at your GitHub Pages URL

---

**Tip:** Start with the Essential Changes section first, then move to Optional Enhancements when ready!
